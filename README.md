- Linux: `~\.config\nvim`
- Windows: `C:\Users\<user>\AppData\Local\nvim`

Utils:
- `gc<leader>`: comment selection 
- `gt`: go to next tab 
- `gT`: go to previous tab
- `gd`: go to definition
- `gf`: go to file (or autoformat)

Comment block:
1. `<C-V>` to select the block
2. `<Shift-I>` to insert before each line
3. `Esc` to apply
